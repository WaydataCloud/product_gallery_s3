Product S3 Gallery

This module allows you to upload product images to Amazon S3 or store them directly as binary attachments in Odoo. The option is configurable from the system settings, giving you flexibility to manage storage based on your infrastructure.

Key Features

  • Upload product images to Amazon S3
  • Store images as binary files in Odoo if S3 is not configured
  • Configurable S3 credentials and bucket from Settings
  • Fallback to binary if S3 is unreachable or misconfigured
  • Simple and clean integration with product views
